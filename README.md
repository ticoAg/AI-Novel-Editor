# AI小说编辑器

一个用于短篇小说创作的AI辅助编辑器，提供智能写作建议和编辑功能。

## 功能特点

- 文本编辑与保存：创建、编辑和保存短篇小说
- AI辅助写作：提供情节建议、角色发展和文风调整等功能
- 用户友好界面：直观的编辑区域和AI功能控制面板

## 技术栈

- 后端：Python (FastAPI)
- 前端：React
- AI集成：OpenAI API

## 安装与运行

### 后端

```bash
cd backend
pip install -r requirements.txt
python server.py
```

### 前端

```bash
cd frontend
npm install
npm start
```

## 使用说明

1. 在编辑区域创建或编辑您的小说
2. 使用右侧面板的AI功能获取写作建议
3. 保存您的作品或导出为不同格式
